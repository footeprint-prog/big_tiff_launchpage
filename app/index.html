Turn on cloud saving

This lets your writing save to the cloud so it follows you across devices. You only do this once per device. It takes about a minute.

  1. Open github.com/settings/personal-access-tokens/new (sign in as the Big Tiff account).
  2. Token name: anything, e.g. "Big Tiff writing".
  3. Expiration: pick the longest option (or "No expiration").
  4. Repository access: choose Only select repositories → pick big-tiff-data.
  5. Permissions: expand Repository permissions → set Contents to Read and write.
  6. Click Generate token, copy it, and paste it below.

Your token is stored only in this browser and is never shared or committed. Without it, your writing still saves locally on this device — it just won't sync to the cloud.
